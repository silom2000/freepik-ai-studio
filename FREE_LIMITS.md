# Freepik Video Tester - Free Limits Feature

## 🆓 Бесплатные лимиты для каждой модели

### Что изменилось

Добавлена информация о бесплатных дневных лимитах для каждой модели генерации видео:

**Image-to-Video модели:**
- **Kling O1 Pro**: 🆓 5 запросов/день
- **Kling O1 Standard**: 🆓 5 запросов/день  
- **Kling 2.0**: 🆓 5 запросов/день
- **WAN 2.6 i2v 720p**: 🆓 20 запросов/день
- **WAN 2.6 i2v 1080p**: 🆓 11 запросов/день

**Text-to-Video модели:**
- **WAN 2.6 t2v 720p**: 🆓 20 запросов/день
- **WAN 2.6 t2v 1080p**: 🆓 11 запросов/день

### UI Обновления

1. **Dropdown с лимитами** - Каждая модель теперь показывает дневной лимит бесплатных запросов
2. **Информационный бейдж** - При выборе модели появляется зелёная плашка с количеством бесплатных запросов
3. **Предупреждение в режиме сравнения** - Показывает сколько запросов будет использовано при тестировании всех моделей

### Как это работает

**Режим одиночного теста:**
- Выберите модель в dropdown
- Увидите информацию: "Free: X requests/day"

**Режим сравнения:**
- При нажатии "Generate Video" появится предупреждение
- Показывает количество моделей и общий дневной лимит
- Например: "⚠️ This will use 5 requests (Total daily free: 56)"

### Технические детали

Лимиты основаны на официальной документации Freepik API:
- Данные хранятся в `src/api/freepik-client.js`
- UI обновления в `src/renderer/app.js`
- Стили в `src/renderer/styles.css`

Это помогает пользователям планировать использование бесплатных запросов и понимать ограничения каждой модели! 🎉
